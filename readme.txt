This Tableau project visualizes information concerning
	world population,
	temperature, and
	carbon dioxiode.

The three data sources and more information concerning them may be found at the following links:
	https://www.kaggle.com/ucsandiego/carbon-dioxide
	https://www.kaggle.com/dbialer/svalbard-climate-19102017
	https://www.kaggle.com/imdevskp/world-population-19602018#population_total_long.csv

There are also three screenshots, showing the interactive Tableau Dashboards which summarizes the findings.
